sendCreatedResult method

void sendCreatedResult (RequestContext req, ResponseContext res, dynamic err dynamic result)

Creates function that sends newly created object as JSON. That function call be called directly or passed as a parameter to business logic components.

If object is not null it returns 201 status code. For null results it returns 204 status code. If error occur it sends ErrorDescription with approproate status code.

  • req a HTTP request object.
  • res a HTTP response object.
